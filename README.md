# OCR — Распознавание текста нейросетью с нуля

Нейросеть на чистом NumPy, которая распознаёт цифры и латинские буквы на изображениях.  
Основана на EMNIST датасете (47 классов). Написана без PyTorch и TensorFlow — только математика.

## Структура проекта

```
├── utils.py       # загрузка датасета EMNIST
├── main.py        # обучение нейросети
├── segment.py     # нарезка картинки на отдельные символы
├── predict.py     # распознавание текста
├── weights_i_h.npy  # веса (появятся после обучения)
├── weights_h_o.npy
├── bias_i_h.npy
└── bias_h_o.npy
```

## Установка

```bash
pip install numpy opencv-python tensorflow matplotlib
```

## Запуск

**Шаг 1 — обучить сеть** (занимает ~10–20 минут):
```bash
python main.py
```

**Шаг 2 — распознать текст на картинке:**
```bash
python predict.py моя_картинка.jpg
```

## Как это работает

1. `utils.py` загружает EMNIST — 112 000 изображений символов 28×28 пикселей
2. `main.py` обучает сеть: **784 → 64 → 47** нейронов, функция активации sigmoid, backpropagation
3. `segment.py` нарезает входную картинку на отдельные буквы через OpenCV (бинаризация + контуры)
4. `predict.py` прогоняет каждую букву через сеть и склеивает результат

## Ограничения

- Работает только с латиницей и цифрами (EMNIST не содержит кириллицу)
- Лучше всего работает с печатным текстом на белом фоне
- Буквы должны не сливаться друг с другом
